# 🎮 RePlay



A high-performance, visually immersive encyclopedia for gamers. Inspired by the clean, data-driven layout of Moctale, **RePlay** provides deep-dives into game mechanics, lore, and stats with a premium, modern UI.



---



## 🛠️ Tech Stack



### **Frontend**

* **ReactJS**: Core library for a reactive, component-based UI.

* **Tailwind CSS v4**: Utilizing the next-gen CSS engine for high-performance styling.

* **Framer Motion**: Powering smooth layout transitions and interactive game-card animations.

* **Lucide React & React Icons**: A unified set of icons for a polished gaming aesthetic.



### **Backend**

* **Python**: Handling logic, API routing, and data processing.

* **MongoDB**: Flexible NoSQL storage for complex game metadata and user collections.



---



## ✨ Key Features



* **Interactive Game Cards**: Smooth hover effects and transitions for a "premium" feel.

* **Dynamic Filtering**: Sort by genre, platform, or developer with real-time updates.

* **Detailed Game Profiles**: Modular sections for system requirements, lore, and screenshots.

* **Dark Mode Native**: Optimized for a high-contrast, eye-friendly gaming aesthetic.

* **Responsive Engine**: Seamless experience across mobile, tablet, and desktop.



---



## 🚀 Getting Started



### **Prerequisites**

* **Node.js** (v20+ recommended)

* **Python** (v3.10+)

* **MongoDB** (Local instance or Atlas)



---



## 🎨 UI/UX Philosophy

* **Fluidity**: Every interaction should feel intentional, using framer-motion for layout shifts.

* **Clarity**: High-density information presented through clean typography and spacing.

* **Performance**: Leveraging Tailwind v4's improved build speeds and zero-runtime CSS.
